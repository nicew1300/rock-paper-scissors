# rock-paper-scissors
This is the first JavaScript project in The Odin Project, a simple rock-paper-scissors game that you can play in the browser.
The hardest part was figuring out the game logic in JavaScript and keeping track of the scores for both the player and the computer. I also had to manage event listeners for the buttons and update the result text dynamically, which was a good practice for DOM manipulation.
In this project, the newly used technologies were event listeners, DOM manipulation, and JavaScript functions to control game state.
